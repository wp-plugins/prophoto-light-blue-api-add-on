== Description ==

### Link your WordPress site to Light Blue 

Light Blue is a business management system designed specifically for professional photographers. If you're using the <a href="http://www.gravityforms.com">Gravity Forms</a> plugin on your WordPress site, you can use this add-on to link your forms directly to your Light Blue account.

####Requirements

To use the Gravity Forms Light Blue API add-on, you need:

* A WordPress site that supports plugins (i.e. not a site that's hosted by WordPress.com).
* GravityForms 1.6 or newer.
* A Light Blue account with a subscription to our online services.

####The Light Blue API

You can find out more about the Light Blue API in <a href="http://www.lightbluesoftware.com/api/">our documentation</a>, which includes more information about how you can go about linking your Gravity Forms to your Light Blue account.

####Support

Any questions about the Gravity Forms Light Blue API add-on can be directed to the <a href="http://www.lightbluesoftware.com/support/">Help section of our website</a>.

== Installation ==

1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Gravity Forms Light Blue API add-on settings page by going to 'Forms' menu, then 'Settings' and then into 'Light Blue API'.
4. Enter your Light Blue API key and click the 'Update Settings' button.
5. Follow the instructions in the settings screen to find out more about the Light Blue API and how to configure Gravity Forms to work with it.
