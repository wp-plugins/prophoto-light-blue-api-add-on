== Description ==

### Link your ProPhoto WordPress site to Light Blue 

Light Blue is a business management system designed specifically for professional photographers. If you're using the <a href="http://www.prophoto.com">ProPhoto 5</a> theme for your WordPress site, you can use this add-on to link your contact form directly to your Light Blue account.

####Requirements

To use the Light Blue API for ProPhoto Plug-in, you need:

* A WordPress site that supports plugins (i.e. not a site that's hosted by WordPress.com).
* The ProPhoto 5 theme.
* A Light Blue account with a subscription to our online services.

####The Light Blue API

You can find out more about the Light Blue API in <a href="http://www.lightbluesoftware.com/api/">our documentation</a>, which includes more information about how you can go about linking your ProPhoto 5 contact form to your Light Blue account.

####Support

Any questions about the Light Blue API for ProPhoto Plug-in can be directed to the <a href="http://www.lightbluesoftware.com/support/">Help section of our website</a>.

== Installation ==

1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Light Blue API for ProPhoto Plug-in settings page by clicking on its 'Settings' link from the 'Installed Plugins' list in WordPress.
4. Enter your Light Blue API key and click the 'Update Settings' button.
5. Follow the instructions in the settings screen to find out more about the Light Blue API and how to configure your ProPhoto 5 contact form to work with it.
